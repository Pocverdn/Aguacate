Acerca de la Aplicación

Esta app fue desarrollada como proyecto final del curso de Análisis Númerico, con el fin de integrar en una sola plataforma todos los métodos vistos a lo largo del semestre. La aplicación está organizada por capítulos según los temas clave: Sistemas de ecuaciones no lineales, Sistemas de ecuaciones lineales e Interpolación.

Cada módulo permite ingresar datos de forma intuitiva, visualizar gráficos, generar tablas de resultados paso a paso, verificar condiciones de convergencia, y generar reportes automáticos de comparación entre métodos. Además, cuenta con un sistema de ayuda para guiar al usuario, prevenir errores comunes y facilitar el uso de funciones algebraicas o derivadas.

Esta herramienta fue creada con Python y Django como backend, lo que permite una estructura robusta, escalable y fácilmente extensible para futuras mejoras.
